# GlobalScreen Privacy
Disclaimer: This documentation does not warrant completeness or correctness. Please report any missing or wrong information using the [ILIAS issue tracker](https://mantis.ilias.de) or contribute a fix via [Pull Request](docs/development/contributing.md#pull-request-to-the-repositories).

## General Information
- The UI component controls the page layout. Components contribute elements to the page layout by using the GlobalScreen component.
- Components which use the GlobalScreen component might store, present, delete or export personal data. This is specified in their respective PRIVACY.md.

## Integrated Services
- The GlobalScreen component employs the following services, please consult the respective privacy.mds:
    - UI

## Data being stored
- The GlobalScreen component does not store any personal data.

## Data being presented
- The GlobalScreen component does not present any personal data.

## Data being deleted
- The GlobalScreen component does not delete any personal data.

## Data being exported
- The GlobalScreen component does not export any personal data.
