## General Information
Advanced Metadata is labeled Custom Metadata in the user interface. 
Advanced Metadata allow adding structured organization-specific data fields to annotate objects. People are supposed add data fields and then to fill in the data fields within objects.  
Once activated the Advanced Metadata are offered in specific object types in the Metadata-tab. 
In categories filters can be activated to filter for Advanced Metadata. 


## Integrated Services
The Advanced Metadata service MUST be listed as integrated service in the Metadata Service > check
The Advanced Metadata service employs the following services, please consult the respective privacy.mds
- AccessControl


## Data being stored
The Advanced Metadata itself does not store any personal data.

## Data being presented
The Advanced Metadata itself does not present any personal data.

## Data being deleted
The Advanced Metadata itself does not store or delete any personal data.

## Data being exported
The Advanced Metadata module itself does not export any personal data.
