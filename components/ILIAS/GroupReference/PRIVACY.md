## General Information
A group reference makes it possible to call up an existing group from a different position.
## Integrated Services
The Group Reference employs the following services, please consult the respective privacy.mds
- AccessControl
- Group
- Membership service
## Data being stored
The Group Reference itself does not store any personal data.
## Data being presented
The Group Reference itself does not present any personal data.
## Data being deleted
The Group Reference itself does not store or delete any personal data.
## Data being exported
The Group Reference itself does not export any personal data.
