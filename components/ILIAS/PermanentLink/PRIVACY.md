# PermanentLink Privacy
Disclaimer: This documentation does not warrant completeness or correctness. Please report any missing or wrong information using the [ILIAS issue tracker](https://mantis.ilias.de) or contribute a fix via [Pull Request](docs/development/contributing.md#pull-request-to-the-repositories).

## General Information
- The PermanentLink component is responsible for generting permanent links 
  to ILIAS objects and some services.

## Data being stored
- The PermanentLink component does not store any personal data.

## Data being presented
- The PermanentLink component does not present any personal data.

## Data being deleted
- The PermanentLink component does not delete any personal data.

## Data being exported
- The PermanentLink component does not export any personal data.
