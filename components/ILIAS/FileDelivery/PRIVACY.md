# FileDelivery Privacy
Disclaimer: This documentation does not warrant completeness or correctness. Please report any missing or wrong information using the [ILIAS issue tracker](https://mantis.ilias.de) or contribute a fix via [Pull Request](docs/development/contributing.md#pull-request-to-the-repositories).

## General Information
- The FileDelivery component is used to deliver files to the browser.
- The FileDelivery component is ignorant of any personal data which might be associated with the delivered files.

## Data being stored
- The FileDelivery component does not store any personal data.

## Data being presented
- The FileDelivery component does not present any personal data.

## Data being deleted
- The FileDelivery component does not delete any personal data.

## Data being exported
- The FileDelivery component does not export any personal data.
