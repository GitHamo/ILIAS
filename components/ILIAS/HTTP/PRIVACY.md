# HTTP Privacy
Disclaimer: This documentation does not warrant completeness or correctness. Please report any missing or wrong information using the [ILIAS issue tracker](https://mantis.ilias.de) or contribute a fix via [Pull Request](docs/development/contributing.md#pull-request-to-the-repositories).

## General Information
- The HTTP component is responsible for handling http requests and responses.
- The HTTP component is ignorant of any personal data contained in these http requests and responses.

## Data being stored
- The HTTP component does not store any personal data.

## Data being presented
- The HTTP component does not present any personal data.

## Data being deleted
- The HTTP component does not delete any personal data.

## Data being exported
- The HTTP component does not export any personal data.
