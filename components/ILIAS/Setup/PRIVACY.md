## General Information

This component contains interfaces, base-classes and utilities which are used by other compoonents for the setup of
ILIAS.

## Integrated Components

The Setup component employs the following components, please consult the respective `PRIVACY.md`:

* [Data](../Data/PRIVACY.md): for representing structured data
* [Refinery](../Refinery/PRIVACY.md): for retrieving and transforming data
* Language: for translations

## Data being stored

This component does not store any personal data.

## Data being presented

This component does not present any personal data.

## Data being deleted

This component does not delete any personal data.

## Data being exported

This component does not export any personal data.
